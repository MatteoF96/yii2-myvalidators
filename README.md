# Yii2 my validators
Dummy validators (only to learn).


## How to install

- Add in `require` of composer.json:
```
"matteo/yii2-myvalidators": "@dev"
```

- Add in `repositories` of composer.json:
```
{
    "type": "vcs",
    "url": "https://github.com/MatteoF96/yii2-myvalidators.git"
}
```

- composer update

- composer install
